How to use this Push and Pull weight loss goal manager

To add a push goal - step 1: Click add push goal and enter the details, then click submit.

To add a push goal - step 2: Repeat until you have all your current push goals listed.

To add a pull goal - step 1: Click add pull goal, enter the details, then click submit.

To add a pull goal - step 2: Repeat until you have all your current push goals listed.

To Delete a goal: If you want to delete a goal then click the red cross next to the goal.

To Edit a goal: If you want to edit a goal, first delete it and then add new with the updated details.

How to use this list: Read this list every day to help inspire and drive you towards your weight loss goals.
